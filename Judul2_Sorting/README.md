Judul : Mengurutkan nama pesrerta lomba lari menggunakan Bubble Sort

Program ini berfungsi untuk mengurutkan nama peserta lomba lari secara alfabet (A–Z) menggunakan algoritma Bubble Sort, di mana pengguna memasukkan jumlah peserta dan nama-namanya lalu disimpan dalam sebuah list. Proses pengurutan dilakukan dengan membandingkan dua nama yang bersebelahan, kemudian menukarnya jika urutannya salah, dan langkah ini diulang terus hingga seluruh data terurut dengan benar. Setelah itu, program menampilkan daftar nama sebelum dan sesudah diurutkan.
